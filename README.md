# KD Financial - Expense Manager 💎

A premium, full-stack financial management system for professional expense tracking, income analysis, and real-time reporting. Designed for high-performance **Local Development** and **Mobile Network** accessibility.

---

## 🚀 Quick Local Setup

Follow these 3 steps to get your project running on your PC and Mobile.

### 1. Database Setup (MySQL)
1. Create a MySQL database named `expense_manager`.
2. Run the `database_setup.sql` script located in the root folder.
3. Update `server/.env` with your MySQL credentials:
```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=expense_manager
```

### 2. Backend Server
```bash
cd server
npm install
node server.js
```
*Server runs on port **3001***.

### 3. Frontend Application
```bash
# In the root folder
npm install
npm run dev -- --host
```
*App runs on port **5174*** (or 5173).

---

## 📱 Mobile & Network Usage

To use your project on your **Phone** or other devices on your Wi-Fi:

1. Find your PC's **IPv4 Address** (Type `ipconfig` in CMD).
2. Ensure your phone is on the **same Wi-Fi** as your PC.
3. Open your mobile browser and enter:
   👉 `http://YOUR_PC_IP:5174`

---

## 🔑 Admin Access
By default, all signups are regular users. To create your first **Super Admin** account:
```bash
cd server
node create_admin.js
```
*Login with the credentials generated by the script.*
